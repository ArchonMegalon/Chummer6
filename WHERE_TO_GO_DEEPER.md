# Where To Go Deeper

![Where to go deeper banner](assets/pages/where-to-go-deeper.png "this is the part where curiosity becomes paperwork with better lighting.")

This page is for when the tour stops being enough and you want the next layer of truth.

Pick the reason you are digging deeper, then follow that lane.

## I want the long-range plan

Go to `chummer6-design` when you want the blueprint: repo ownership, split order, milestone framing, and the longer-range direction behind the public guide.

## I want the actual software

Go to the owning code repos when you want the implementation, build steps, repo-specific readmes, and the real code that powers the visible surfaces.

## I want to check why a rule behaved that way

Start with the engine-side material: core instructions, explain receipts, provenance trails, and the docs that describe deterministic evaluation.

If the question is "why did the math do that?", this is your lane.

## I want to bend the rules for my table

Follow the Lua and era-specific materials when you want custom behavior, scripted edge logic, or the proof behind multi-era and SR4-style support.

If the question is "how do I make my table's weirdness legal?", this is your lane.

## I want to understand what people actually use

Read the UI and play readmes when you care about the visible surfaces across desktop, browser, and table-facing session flow.

If the question is "what do users actually click?", this is your lane.
